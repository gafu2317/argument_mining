# 再利用パターン

このファイルにはプロジェクトで繰り返し使用するパターンを記録します。

## フォーマット

```markdown
## パターン名

**用途**: このパターンをいつ使うか

**実装**:
[コード例や手順]

**注意点**: 注意すべきこと

**関連**: 関連する他のパターン
```

---

## プロジェクトの基本構造

**用途**: 新しいプロジェクトを開始するとき

**実装**:
```
.
├── AGENTS.md           # 開発フロー概要
├── CLAUDE.md           # Claude Code 設定
├── Plans.md            # タスク管理
├── .claude/
│   ├── settings.json   # 権限・安全設定
│   ├── memory/         # 意思決定記録・パターン
│   └── rules/          # 品質保護ルール
```

**注意点**:
- 既存ファイルは上書きしない
- 非破壊マージで設定を追加

**関連**: decisions.md（意思決定の記録）
